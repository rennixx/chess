# chess.krd

A mobile-first chess platform built with Expo, targeting Kurdish-speaking users.

## Getting Started

### Frontend (Expo)
```bash
cd app
npm install
npx expo start
```

### Backend (Go)
```bash
go mod tidy
go run cmd/api/main.go
```

## Tech Stack
- **Frontend:** Expo SDK 54, TypeScript, chess.js, Zustand, NativeWind
- **Backend:** Go, Chi router, PostgreSQL, JWT auth
- **Engine:** Client-side AI (random/material/minimax)

## Languages
- English (primary)
- Sorani (کوردی)
- Kurmanji (Kurmancî)
